**Nazzava** adalah platform web bertema **lingkungan** untuk mengedukasi, mengukur, dan mengajak pengguna beraksi menjaga bumi.  
Fokus utama Nazzava adalah memberi alat praktis (kalkulator emisi, pengenal sampah, informasi elemen lingkungan, permainan) serta kanal **chatbot AI** untuk bertanya seputar eco-lifestyle.

Dibangun dengan **Next.js + TypeScript** sebagai framework utama, **Google Generative AI** untuk chatbot, dan **Tailwind CSS** untuk tampilan yang modern dan responsif.
